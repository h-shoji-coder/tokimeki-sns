# ときめき（Tokimeki）

## 概要
婚活に特化した「バズる」SNSアプリ。相性診断・スワイプマッチング・タイムライン・ランキングなどSNS的な拡散要素を組み合わせ、出会いと話題を同時に提供する。

## 解決する課題
- 婚活アプリは「真剣すぎて疲れる」「続かない」という離脱問題
- マッチングしても会話が弾まない
- 婚活していることをSNSで自然にシェアしにくい

## ターゲットユーザー
25〜35歳の婚活中の男女。SNSヘビーユーザーで、TikTok・Instagramに慣れ親しんでいる層。

## 主要機能
- 恋愛・相性診断クイズ（シェアしたくなる診断結果）
- スワイプ型プロフィールカード（Like / Pass）
- 婚活タイムライン（投稿・コメント・いいね）
- マッチングランキング・トレンド
- 共通点カウンター（プロフィール比較）
