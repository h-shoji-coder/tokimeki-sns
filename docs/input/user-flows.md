# ユーザーフロー

## 1. オンボーディングフロー

### 新規登録
1. トップページで「無料で始める」をタップ
2. SNSログイン（Google/LINE）またはメールで登録
3. 基本プロフィール入力（名前・年齢・写真）
4. 恋愛タイプ診断（5問のクイック版）
5. ホームタイムラインへ遷移

### 初回利用
1. 「今日のおすすめ」カード表示
2. スワイプして気になる相手にいいね
3. マッチング成立 → お祝い演出 → トーク開始促進

## 2. メインフロー

### 相性診断
**目的：** 自分の恋愛タイプを知り、SNSでシェアしたくなる

**フロー：**
1. ホームから「診断する」ボタンをタップ
2. 10問の選択式質問に回答（1問あたり4択）
3. 診断結果ページ表示（タイプ名・説明・アドバイス）
4. 「シェアする」ボタンでSNSシェア用カード画像生成
5. シェア or 「気になる人の診断結果を見る」へ

**成功条件：** 診断完了 → シェア

---

### スワイプマッチング
**目的：** 気になる相手とマッチングする

**フロー：**
1. 「見つける」タブを開く
2. カード形式でプロフィールを表示（写真・名前・年齢・共通点数）
3. 右スワイプ or ハートボタンでいいね
4. 左スワイプ or ×ボタンでパス
5. 相手もいいねしていたらマッチング成立アニメーション
6. 「メッセージを送る」またはタイムラインに戻る

---

### タイムライン投稿
**目的：** 婚活日常をシェア・共感を得る

**フロー：**
1. 「タイムライン」タブを開く
2. 他ユーザーの投稿を閲覧（いいね・コメント）
3. 「＋」ボタンをタップして投稿作成
4. テキスト入力 or 画像選択 + ハッシュタグ追加
5. 投稿 → タイムラインに反映

## 3. その他のフロー

### ログイン
1. アプリ起動
2. SNSログインまたはメール・パスワード入力
3. ホームへ遷移

### プロフィール編集
1. マイページタブを開く
2. 「編集」ボタンをタップ
3. 写真・自己紹介・趣味などを更新
4. 保存 → プロフィール画面へ
