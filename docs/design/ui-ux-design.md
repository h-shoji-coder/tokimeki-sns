# UI/UX設計書 - ときめき（Tokimeki）婚活SNS

## 1. デザインコンセプト

### 1.1 カラーパレット
- **Primary**：ローズピンク `#F43F5E`
- **Primary Light**：`#FB7185`
- **Accent**：ゴールド `#FBBF24`（ランキング・特別感）
- **Background**：オフホワイト `#FFF1F2`
- **Surface**：ホワイト `#FFFFFF`
- **Text Primary**：`#1F2937`
- **Text Secondary**：`#6B7280`
- **Border**：`#FCE7F3`

### 1.2 タイポグラフィ
- Font Family：`'Noto Sans JP', 'Inter', sans-serif`
- 見出し：Bold・丸みのあるゴシック体
- 本文：Regular・16px・行間1.6

### 1.3 デザイン方針
- モバイルファースト・縦スクロール中心
- カード型UIで情報を整理
- アニメーションを積極使用（ときめき演出）
- 余白を十分に取り、圧迫感を出さない

## 3. 主要画面ワイヤーフレーム

### ランディングページ
- **ヒーローセクション**：「出会いをもっと、ときめきに。」キャッチコピー + CTA ボタン
- **診断誘導**：「まず恋愛タイプを診断する」CTA（バイラル入口）
- **特徴紹介**：3カラムのアイコン + テキスト
- **人気ユーザープレビュー**：ぼかし加工で雰囲気を見せる
- **フッター**：規約・お問い合わせリンク

### ホーム（タイムライン）
- **ヘッダー**：ロゴ + 通知ベル + メッセージアイコン
- **ストーリー**：横スクロールの丸型アイコン列
- **投稿フィード**：カード型・写真/テキスト・いいね/コメントボタン
- **FAB（浮き丸ボタン）**：投稿作成ボタン

### 見つける（スワイプ）
- **カード**：相手写真フルカバー + 名前・年齢・共通点バッジ
- **操作ボタン**：左（パス×）・右（いいね♡）の2択
- **共通点チップ**：趣味・価値観の一致アイテム表示

### 恋愛診断
- **クイズ画面**：プログレスバー + 質問カード + 4択ボタン
- **結果画面**：タイプ名・絵文字・説明・シェアボタン

### ランキング
- **週間いいね数ランキング**：TOP10 ユーザーカード
- **トレンドハッシュタグ**：今週の急上昇タグ
- **コミュニティ統計**：今週のマッチング成立数アニメーション
